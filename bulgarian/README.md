# Bulgarian language

### Произношение 

- все звуки читаются как пишутся, кроме 
  - “е” читается как “э”
  - “ъ” читается как “ы”
  - “ж” и “ш” читаются как “ж с хвостиком” и “щ”
  - “щ” читается как “шт”
- согласный звук в конце слова всегда твердый (“конь”, читается “кон”)



### Слова

#### Местоимения
- я -> аз
- ты -> ти
- он -> той
- она -> тя
- мы -> ние
- вы -> вие

#### Вопросы
- что -> какво
- кто -> кой

#### Время
- сейчас -> сега 
- всегда -> в**и**наги
- никогда -> н**и**кога

#### Местоположение
- здесь -> тук
- там, туда -> там

#### Глаголы 
- делаешь -> правиш (что делаешь? -> какво правиш?)
- работаю -> работя

#### Существительные
- время -> време
- это -> това

#### Отрицательная форма
- Я не ... -> Аз **не** съм ...

#### Вопросительная форма
- Я? -> Аз **ли** съм?

### Предложения

- Я студент -> Аз **съм** студент
